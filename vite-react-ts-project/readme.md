### 1 `npm install`

install node 18 first, then run npm install to install the project

### 2 `npm run dev`

run dev project

### 3 `npm install -D tailwindcss postcss autoprefixer`

install tailwind in react

### 4 `npx tailwindcss init -p`

init tailwind


### 5 `npm i json-server`

Install json server

### 6 `npm run server`

Run json server
